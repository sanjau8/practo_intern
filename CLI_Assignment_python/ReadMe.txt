========== PYTHON SCRIPT TO FETCH WIKIPEDIA LINKS BASED ON KEYWORD===============


Directions to run

1. open terminal and move to the folder where getWikiLink.py is contained
2. check if requests package is installed
3. run the command python -m [--mode] [0 | 1 | cli | interactive] [-v | --value] [seachTerm]
4. [-v | --value] is must if the mode is [0 | cli]
5. if mode [1 | interactive] is chosen, a prompt to enter search term will be asked
